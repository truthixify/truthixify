# Me 🤓

I'm a mathematician, software developer, and cryptographer.  

I specialize in creating software such as developer tools, websites, cryptographic protocols, and distributed systems.  

I hold a strong philosophical belief that privacy and anonymity are fundamental rights. I am convinced that cryptography can make this possible—so, yes, you could call me a cypherpunk.  

## Interests:  
- **Programming:** Distributed systems, website development  
- **Mathematics:** Number theory, abstract algebra, and recreational mathematics  
- **Cryptography:** Fully Homomorphic Encryption (FHE), Zero-Knowledge proofs (ZK), and secure implementation of cryptographic algorithms in Rust  
- **Movies:** Anime, crime dramas, and classic Bollywood films  

## Notable Projects: 
- **[Chip8](https://github.com/truthixify/chip8):** A Chip8 emulator written in Rust, integrated into the browser using WebAssembly (WASM).  
- **[Scheme Interpreter](https://github.com/truthixify/scheme-interpreter):** An interpreter for the Scheme programming language, implemented in Rust.  
- **MD6:** A pure Rust implementation of the MD6 cryptographic hash function.  
- **[SHA](https://github.com/truthixify/sha):** A Rust-based implementation of the SHA family of cryptographic algorithms.  
- **[Cask-DB-RS](https://github.com/truthixify/cask-db-rs):** A Rust implementation of the Bitcask key-value database.
